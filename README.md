# Thiệp chúc mừng sinh nhật (static)

Tệp này là một thiệp sinh nhật nhỏ bằng HTML/CSS/JS. Mở `index.html` trong trình duyệt để xem.

Hướng dẫn nhanh để chia sẻ (GitHub Pages):
- Tạo repo trên GitHub, push toàn bộ nội dung workspace.
- Vào Settings → Pages → Source → chọn `main` branch và root. Lưu.
- Sau vài phút, trang sẽ có dạng `https://<your-username>.github.io/<repo>` và người yêu bạn có thể truy cập.

Nếu muốn, mình có thể giúp deploy lên GitHub Pages luôn.
